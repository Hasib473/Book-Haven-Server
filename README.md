# 📚 The Book Haven (Server)

This is the backend server for **The Book Haven** web application.  
It handles API requests, manages book data, and connects with the database.

🌐 **Live Server:**  
https://book-haven-server-beige.vercel.app/

---

## 🚀 Features

- 📚 REST API for managing books
- ➕ Add, update, and delete book data
- ⭐ Store ratings and user related data
- ⚡ Fast API responses with Express.js
- 🌍 Deployed server accessible globally

---

## 🛠️ Technologies Used

- Node.js
- Express.js
- MongoDB
- CORS
- dotenv

---
